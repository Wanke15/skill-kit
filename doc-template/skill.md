# 文档模板 Skill

将结构化内容转换为美观的HTML文档，支持导航栏、卡片布局、时间线等多种组件。

## 功能

- 生成带有粘性导航栏的HTML文档
- 支持多种内容组件：卡片网格、时间线、定义列表、代码块、对比框等
- 响应式设计，适配移动端和桌面端
- 支持中文排版优化

## 使用方法

```
/doc-template [markdown内容或主题描述]
```

## 可用组件

### 基础组件
- `highlight-box` - 高亮提示框（黄色背景，左侧强调线）
- `info-box` - 信息提示框（蓝色背景，左侧信息线）
- `card` / `cards-grid` - 卡片网格布局
- `code-block` - 代码块（深色主题）
- `table` - 表格样式

### 导航组件
- `nav` - 粘性导航栏，自动高亮当前章节
- `toc` - 目录组件（双列布局）

### 布局组件
- `definition-list` / `definition-item` - 定义列表（左侧强调线）
- `step-list` - 步骤列表（带数字圆圈）
- `timeline` / `timeline-item` - 时间线（垂直线+节点）
- `comparison-box` - 对比框（并排比较）

### 标签组件
- `tag` - 通用标签
- `method-badge` - 方法标签（支持 code/model/human 三种风格）

## 输出格式

生成独立的HTML文件，可直接在浏览器中打开，无需外部依赖。

## 示例

输入：
```
/doc-template
主题：AI评估方法论
章节：引言、核心概念、实施步骤
```

输出：
完整的HTML文档，包含样式、导航和结构化内容。
